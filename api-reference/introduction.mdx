API ini menyediakan endpoint untuk mengelola sistem komentar dan balasan berbasis link unik (\`link_id\`). Cocok digunakan untuk aplikasi atau sistem yang membutuhkan sistem pesan anonim, kolom komentar, atau feedback publik.

## Base URL

\`\`\`
https://api.kisara.my.id
\`\`\`

## Autentikasi

Untuk beberapa endpoint (seperti menghapus komentar atau memberi like), autentikasi JWT diperlukan. Token dapat diperoleh melalui proses login di endpoint otentikasi (\`/auth/login\`).

Gunakan token JWT dalam header:

\`\`\`
Authorization: Bearer <your_token>
\`\`\`

## Rate Limiting

Semua endpoint memiliki batas maksimal **30 request per menit per IP** untuk mencegah spam.

## Format Response

Setiap response akan dikembalikan dalam format JSON berikut:

\`\`\`json
{
"statusCode": 200,
"message": "Success",
"data": {
...
}
}
\`\`\`

Jika terjadi error:

\`\`\`json
{
"statusCode": 404,
"message": "User with link_id xyz not found",
"data": null
}
\`\`\`

## Endpoint Groups

- **Auth**: login, register, get profile.
- **Message**: kirim komentar, balas komentar, hapus, like.
