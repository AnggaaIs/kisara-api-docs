## Environment

- Base URL (Production): `https://api.kisara.my.id`
- OpenAPI JSON: `https://api.kisara.my.id/openapi.json`
- OpenAPI YAML: `https://api.kisara.my.id/openapi.yaml`

## Alur Integrasi

<Steps>
	<Step title="Dapatkan Access Token">
		Lakukan autentikasi sesuai endpoint yang tersedia, lalu simpan token JWT Anda.
	</Step>
	<Step title="Kirim Request ke Endpoint">
		Sertakan header `Authorization: Bearer <access_token>` untuk endpoint yang memerlukan autentikasi.
	</Step>
	<Step title="Validasi Respons">
		Cek `status_code`, `message`, dan isi `data`. Gunakan `metadata` untuk troubleshooting.
	</Step>
</Steps>

<Tip>
	Untuk pengalaman terbaik, buka halaman [API Overview](/api/overview) terlebih dahulu lalu lanjut ke API Reference.
</Tip>
